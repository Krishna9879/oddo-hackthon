====================================================================
TROUBLESHOOTING
====================================================================

Problem: "npm is not recognized"
Solution: Install Node.js from nodejs.org first

Problem: "MySQL connection error"
Solution: 
   - Make sure MySQL is running in XAMPP
   - Check database name is exactly 'workzen'
   - Verify .env.local has correct credentials

Problem: "Port 3000 already in use"
Solution: Run instead: npm run dev -- -p 3001

Problem: "Module not found"
Solution:
   1. Delete node_modules folder
   2. Delete package-lock.json
   3. Run npm install again

Problem: "Can't login"
Solution:
   - Make sure schema.sql was imported
   - Check 'users' table has admin entry
   - Password is: admin123 (not Admin123)

Problem: "Database schema not imported"
Solution:
   - Re-import schema.sql in phpMyAdmin
   - Make sure you selected 'workzen' database first
   - Copy entire file content, don't import file
